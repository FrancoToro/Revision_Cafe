
Introduccion
Cafemixes es una aplicación móvil desarrollada para los amantes del café de cualquier nivel, ya sea amateur o profesional, que buscan compartir y personalizar recetas de café. Esta app nos permitirá explorar una amplia variedad de recetas, que van desde las más elaboradas hasta las más sencillas, adaptándose a cada usuario. Además, los usuarios no solo pueden buscar, sino también subir sus propias recetas, creando una comunidad activa y centrada en el café.
Requerimientos Funcionales:

La aplicación permite un listado de recetas personalizables.
La aplicación tiene la capacidad de guardar recetas como favoritas.
La aplicación permite crear guías predefinidas y adaptables a los diferentes tipos de café.
La aplicación permite al usuario modificar las recetas según su preferencia.
La aplicación permite que el usuario pueda iniciar sesión (loguearse).
La aplicación permite una interacción activa con la comunidad, permitiendo a los usuarios comentar y calificar las recetas.

Requerimientos No Funcionales:

La aplicación está diseñada para dispositivos móviles.
La interfaz es fácil de usar.
La aplicación tiene la capacidad de manejar una gran cantidad de recetas.
La aplicación es escalable, permitiendo el manejo de un gran número de usuarios.
